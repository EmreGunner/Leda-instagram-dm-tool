# Database (Required)
# ⚠️ CRITICAL: Must start with postgresql:// or postgres://
# Get from: Vercel Postgres, Supabase, Railway, or other provider
# 
# Correct format examples:
#   postgresql://user:password@host:5432/database
#   postgresql://user:password@host.example.com:5432/database
#   postgres://user:password@host:5432/database
#
# ❌ WRONG (will cause errors):
#   user:password@host:5432/database  (missing protocol)
#   postgresql:user:password@host     (wrong format)
#   DATABASE_URL= postgresql://...    (extra spaces)
#
DATABASE_URL=postgresql://user:password@host:port/database
DIRECT_URL=postgresql://user:password@host:port/database

# ============================================
# HOW TO GET VALUES:
# ============================================
# 
# DATABASE_URL & DIRECT_URL:
#   - Vercel: Dashboard → Storage → Create Postgres → Copy URL
#   - Supabase: Dashboard → Settings → Database → Connection string
#   - Railway: PostgreSQL Service → Variables → DATABASE_URL
#
# NEXT_PUBLIC_SUPABASE_URL & NEXT_PUBLIC_SUPABASE_ANON_KEY:
#   - Supabase Dashboard → Settings → API
#   - Copy "Project URL" and "anon public" key
#
# JWT_SECRET & ENCRYPTION_KEY:
#   - Run: openssl rand -base64 32
#   - Generate two different values
#
# NEXT_PUBLIC_BACKEND_URL:
#   - Set after first deployment
#   - Use your Vercel project URL
#
# REDIS_URL:
#   - Upstash (Recommended): 
#     1. Sign up at https://upstash.com (free)
#     2. Create a Redis database
#     3. Copy the "REST API" URL (or "Redis URL" from connection details)
#     4. Format: redis://default:PASSWORD@HOST:PORT
#   - Redis Cloud:
#     1. Sign up at https://redis.com/try-free/
#     2. Create a free database
#     3. Copy the connection string
#   - Railway:
#     1. Create a new Redis service
#     2. Copy the REDIS_URL from variables
#
# GEMINI_API_KEY:
#   - Go to: https://makersuite.google.com/app/apikey
#   - Create API key (free tier: 1,500 requests/day)
#   - Copy and paste here
#
# Instagram Service Account (for Instagram data tools):
#   - Create a dedicated Instagram account (e.g., @yourbrand_tools)
#   - Login to Instagram in browser
#   - Open DevTools → Application → Cookies
#   - Copy these cookies:
#     * sessionid → INSTAGRAM_SERVICE_SESSION_ID
#     * ds_user_id → INSTAGRAM_SERVICE_DS_USER_ID
#     * csrftoken → INSTAGRAM_SERVICE_CSRF_TOKEN
#   - Optional: ig_did, mid, rur
#   - Note: Cookies expire every few weeks, need to refresh
#
